LangSplainer

A small toolbox for language learners: sentence explainers, difficulty analysis, text splitting, and Anki workflows.

Most tools use your own API key and store it locally in localStorage. Anki tools expect Anki + AnkiConnect running on your machine.

Sentence Explainers

Sentence Explainer

OpenAI

Multi-angle explanation and JSON output: simplified versions, translations, segmented breakdown, and recommendations.

Sentence Explainer (CN)

Gemini

Chinese‑first breakdown with pinyin, inline grammar tags, and compact HTML formatting.

Sentence Explainer (Experimental)

GPT‑5

Experimental variant of the explainer flow. Useful for comparison and prompts exploration.

Interactive Explainer

OpenAI

Select text inside a passage to get an immediate, focused breakdown of the selected snippet.

Anki + Gemini

Anki Sentence Miner

Segment long text, review/edit sentences, then add cards directly to Anki via AnkiConnect.
